## Get ready

Install [node.js](http://nodejs.org/) then run from the command line

```sh
npm install
```

Check your install: run the test

```sh
npm test
```

## Run the app

from your command line

```sh
npm start
```

Open your browser and navigate to

```sh
http://localhost:8080
```
